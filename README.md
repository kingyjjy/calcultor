## 📢소개📢
react-native와 expo를 사용하여 다크모드와 라이트모드를 바꿀 수 있는 계산기 앱을 만들어 보았습니다.

## 🔎How to🔎
- zip파일 다운로드
- 스마트폰에서 expo go어플을 다운받거나, pc에서 안드로이드 스튜디오 설치 후
- npm start 한 후
- QR코드 expo go에서 보기 or 터미널에서 a 클릭후 안드로이드 스튜디오로 보기
